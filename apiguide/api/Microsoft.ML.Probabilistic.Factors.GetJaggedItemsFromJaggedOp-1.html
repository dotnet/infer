    The type of the resulting array.
    Remarks

    The outgoing message is the exponential of the average log-factor value, where the average is over all arguments except array. Because the factor is deterministic, items is integrated out before taking the logarithm. The formula is exp(sum_(indices,indices2) p(indices,indices2) log(sum_items p(items) factor(items,array,indices,indices2))).
