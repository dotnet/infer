public static class SoftmaxOp_BL06_LBFGS
    Remarks

    This implementation uses the simple first order Taylor series expansion from Blei and Lafferty (2006), followed by optimization using LBFGS. This approach is linear in the dimension K.

    Remarks

    The outgoing message is a distribution matching the moments of softmax as the random arguments are varied. The formula is proj[sum_(x) p(x) factor(softmax,x)].
