    Declaration
    public static ArrayType ToArray<ArrayType>(object distributionArray)
    Parameters
    Type Name Description
    Object distributionArray

    A distribution over an array domain, such as IDistribution<bool[]> or IDistribution<double[,][]>.
    Returns
    Type Description
    ArrayType

    An array of element distributions.
    Type Parameters
    Name Description
    ArrayType

    A .NET array type, such as Bernoulli[] or Gaussian[,][]. The array structure should match the domain of the distribution.
