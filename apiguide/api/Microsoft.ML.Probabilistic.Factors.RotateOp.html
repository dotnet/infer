    Parameters
    Type Name Description
    VectorGaussian rotate

    Incoming message from rotate. Must be a proper distribution. If any element is uniform, the result will be uniform.
    Gaussian x

    Incoming message from x. Must be a proper distribution. If uniform, the result will be uniform.
    Gaussian y

    Incoming message from y. Must be a proper distribution. If uniform, the result will be uniform.
    Returns
    Type Description
    WrappedGaussian

    The outgoing VMP message to the angle argument.
    Remarks

    The outgoing message is the exponential of the average log-factor value, where the average is over all arguments except angle. Because the factor is deterministic, rotate is integrated out before taking the logarithm. The formula is exp(sum_(x,y) p(x,y) log(sum_rotate p(rotate) factor(rotate,x,y,angle))).

    Exceptions
    Type Condition
    ImproperMessageException

    rotate is not a proper distribution.

    x is not a proper distribution.

    y is not a proper distribution.
