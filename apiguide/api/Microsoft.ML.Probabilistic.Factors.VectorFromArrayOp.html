    where GaussianList : IList<Gaussian>

    Type Name Description
    VectorGaussian vector

    Incoming message from fromArray. Must be a proper distribution. If any element is uniform, the result will be uniform.
    IList<Gaussian> array

    Incoming message from data.
    GaussianList result

    Modified to contain the outgoing message.
    Returns
    Type Description
    GaussianList

    result
    Type Parameters
    Name Description
    GaussianList

    The type of the outgoing message.
    Remarks

    The outgoing message is a distribution matching the moments of data as the random arguments are varied. The formula is proj[p(data) sum_(fromArray) p(fromArray) factor(fromArray,data)]/p(data).

    Exceptions
    Type Condition
    ImproperMessageException

    vector is not a proper distribution.

    Returns

    Remarks
