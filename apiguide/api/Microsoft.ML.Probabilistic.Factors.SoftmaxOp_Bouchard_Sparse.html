    Class SoftmaxOp_Bouchard_Sparse

    Provides outgoing messages for Softmax(IList<Double>), given random arguments to the function.

    Inheritance
    Object
    SoftmaxOp_Bouchard_Sparse
    Namespace: Microsoft.ML.Probabilistic.Factors
    Assembly: Microsoft.ML.Probabilistic.dll
    Syntax
    public static class SoftmaxOp_Bouchard_Sparse : object
    Remarks

    This implementation uses the simple first order Taylor series expansion from Blei et al. 06, followed by optimization using LBFGS. This approach is linear in the dimension K.

    Remarks

    The outgoing message is a distribution matching the moments of softmax as the random arguments are varied. The formula is proj[sum_(x) p(x) factor(softmax,x)].
