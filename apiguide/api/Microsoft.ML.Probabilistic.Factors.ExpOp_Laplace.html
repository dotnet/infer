    Double x

    Buffer x.

    Type Description
    Gaussian

    The outgoing EP message to the d argument.
    Remarks

    The outgoing message is a distribution matching the moments of d as the random arguments are varied. The formula is proj[p(d) sum_(exp) p(exp) factor(exp,d)]/p(d).
