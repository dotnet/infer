    Remarks
        The distribution is estimated via moment matching (not maximum-likelihood).
    In the one-dimensional case,
    E[x] = (a+1)/b
    var(x) = (a+1)/b^2
    b = E[x]/var(x)
    a = E[x]^2/var(x) - 1
