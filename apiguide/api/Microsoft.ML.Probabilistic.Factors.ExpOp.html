    DNonconjugateAverageLogarithm(Gamma, Gaussian, NonconjugateGaussian)

    Nonconjugate VMP message to 'd'.

public static NonconjugateGaussian DNonconjugateAverageLogarithm(Gamma exp, Gaussian d, NonconjugateGaussian result)
    Parameters
    Type Name Description
    Gamma exp

    Incoming message from 'exp'. Must be a proper distribution. If uniform, the result will be uniform.
    Gaussian d
    NonconjugateGaussian result
    Returns
    Type Description
    NonconjugateGaussian

    The outgoing nonconjugate VMP message to the 'd' argument.
    Remarks

    The outgoing message is the exponential of the integral of the log-factor times incoming messages, over all arguments except 'd'. The formula is int log(f(d,x)) q(x) dx where x = (exp).
